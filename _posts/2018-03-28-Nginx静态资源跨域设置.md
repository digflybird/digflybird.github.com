在nginx的配置文件 `conf/nginx.conf` 中增加头部信息，具体如下所示:

```
http {
    ...

    server {
        listen       80;
        server_name  static.mydomain.cn;

        location /static {
            root html;
            add_header Access-Control-Allow-Origin *; # 你需要增加的 HTTP 头
        }
    }
}
```
